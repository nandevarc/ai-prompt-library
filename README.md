# AI Prompt Library
A collection of useful AI prompts for productivity, learning, and content generation.

This repository contains a curated collection of useful prompts for working with AI (LLMs).

## Categories

### 1. Productivity

**Task Breakdown**
"Break down this task into simple actionable steps: [TASK]"

**Focus Assistant**
"Act as a productivity coach. Help me stay focused on [TASK] and suggest the next best action."

---

### 2. Learning

**Explain Like I'm 5**
"Explain [TOPIC] in the simplest way possible like I'm 5 years old."

**Deep Understanding**
"Explain [TOPIC] step by step with examples and practical use cases."

---

### 3. Problem Solving

**Solution Generator**
"Give me 3 practical solutions for this problem: [PROBLEM]"

**Decision Helper**
"Help me decide between these options: [OPTION A] vs [OPTION B] with pros and cons."

---

### 4. Content (Optional)

**Simple Content Idea**
"Generate 5 simple content ideas about [TOPIC]"

---
